# Default Strait of Hormuz bounding box

The script defaults to this bounding box (override via env vars):

- `HORMUZ_LAT_MIN=25.5`
- `HORMUZ_LAT_MAX=27.5`
- `HORMUZ_LON_MIN=56.0`
- `HORMUZ_LON_MAX=57.5`

Adjust it if you want:
- wider coverage (approaches / holding areas)
- a tighter “choke point” focus
